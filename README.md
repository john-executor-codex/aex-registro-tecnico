# AEX Registro Técnico

Sistema de registro técnico da **AEX | Inteligência Construtiva**, com deploy via GitHub Pages.

## Sobre

Este repositório concentra o registro técnico oficial da AEX, incluindo documentação, especificações e referências de projetos desenvolvidos com IA aplicada à construção civil.

O site está disponível em: [https://john-executor-codex.github.io/aex-registro-tecnico](https://john-executor-codex.github.io/aex-registro-tecnico)

## Contexto do Projeto

A **AEX Inteligência Construtiva** desenvolve soluções de IA para o setor de construção, com foco em:

- Automação BIM com Revit e Python
- Integração de GPT Builder para assistentes especializados
- Ferramentas de gestão técnica e documentação inteligente
